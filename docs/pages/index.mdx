# LiL Grid

### A tiny composable grid for styled-components.

Built for ease of use and low overhead. Use this for any and all of your react projects.

Install with your favorite package manager.

npm install lil-grid styled-components

## Basic Usage

This grid is built with two components. `Grid` is a wrapper and `Col` is always an immediate child of the `Grid`.

<br />
<br />

### Array prop syntax

The column component accepts two props to style the grid. Both should be passed as arrays, the arrays represent changes at each breakpoint: [xsm, sm, md, lg, xlg]

<br />
<br />
<br />
<section>
  {rows.map(row => (
    <Grid style={{marginBottom: "2rem"}}>
      {row.map(col => <Col
          span={[12, col / 2, col]}
          style={style}>
            <p style={{fontSize: 12, margin: 0}}>{col}</p>
        </Col>
      )}
    </Grid>
  ))}
</section>

```jsx
import { Col, Grid } from "lil-grid"

const style = { background: colors.blue_100, padding: "2rem" }

function Component() {
  return rows.map(row =>
    row.map(col => (
      <Col span={[12, col]} style={style}>
        <p style={{ fontSize: 12, margin: 0 }}>{col}</p>
      </Col>
    ))
  )
}
```

## Grid

| Name  | Required | Default     | Description                                                   |
| ----- | -------- | ----------- | ------------------------------------------------------------- |
| theme | no       | ThemeObject | Pass as a theme object to customize grid breakpoints and gaps |

## Col

| Name | Required | Default | Description                             |
| ---- | -------- | ------- | --------------------------------------- |
| span | yes      | n/a     | Array of amounts for the column to span |
